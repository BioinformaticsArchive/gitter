Installation

To begin using gitter download the R package and run the following in command line:

$ R CMD INSTALL gitter_1.0.tar.gz

or from an R shell

$ install.packages('gitter_1.0.tar.gz', repos=NULL)

Getting started

To get started, load the gitter package:

$ install.packages('gitter_1.0.tar.gz', repos=NULL)

Process a single sample image:

# Read sample image path
f = system.file("data", "sample.jpg", package="gitter")
# Process it
dat = gitter(f)
# View head of the results
head(dat)

# Remember:
# output files by default are saved to your working directory

For further documentation on the parameters you can pass to gitter or how to process batch images try typing the following into your R console:

?gitter
?gitter.batch

For information on the package:

?Rgitter
